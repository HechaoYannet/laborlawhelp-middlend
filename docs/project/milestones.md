# Milestones (M1/M2/M3)

## M1 - Main Flow Ready
Goal: unblock parallel frontend/backend development.

Deliverables:
- API contract markdown
- Error code manual
- Frontend integration guide
- Backend implementation blueprint

Exit criteria:
- Contract review approved
- Stream sequence and error handling agreed
- Frontend and backend kickoff complete

## M2 - Data and Ops Ready
Goal: ensure dev/staging operation and migration readiness.

Deliverables:
- DB schema and migration doc
- Redis strategy doc
- Environment runbook
- Observability doc
- Deploy and rollback doc
- Anonymous-to-user migration template

Exit criteria:
- Dev and staging smoke pass
- Alerts and dashboard configured
- Migration rehearsal passes in staging snapshot

## M3 - Quality Gate Ready
Goal: define release gate by test and acceptance.

Deliverables:
- Test plan with performance baseline
- Acceptance checklist
- Unified glossary
- Documentation index and role-based reading path

Exit criteria:
- Functional + stream + failure tests pass
- Performance baseline met
- Documentation sign-off from FE/BE/QA/Ops
